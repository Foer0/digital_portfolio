# Цифровой портфель

Веб-платформа для поиска работы и сотрудников с функционалом портфолио.

## Функционал

### Для соискателей:
- Создание и редактирование портфолио
- Поиск вакансий с фильтрацией
- Отклик на вакансии с сопроводительными письмами
- Отслеживание статуса откликов

### Для работодателей:
- Создание компании и вакансий
- Просмотр портфолио соискателей
- Управление откликами
- Модерация вакансий

### Для администраторов:
- Управление пользователями и компаниями
- Модерация вакансий
- Просмотр статистики

## Технологии

- **Backend:** Flask, SQLAlchemy
- **Frontend:** HTML, CSS, JavaScript, Bootstrap 5
- **Database:** SQLite
- **Аутентификация:** Flask-Login
